# 💬 Chat Application (MERN Stack)

A real-time Chat Application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) with Cloudinary integration for profile picture uploads.
This app allows users to sign up, log in, create groups, search users, and chat in real time.

# 🚀 Features

🔐 User Authentication (Login & Signup with profile picture)

👥 One-to-One Chat & Group Chat

🔎 Search Users to start chatting instantly

📸 Profile Picture Upload using Cloudinary

📨 Real-time Messaging

🔔 Notifications for new messages

💻 Responsive UI

🛠 Tech Stack

Frontend: React.js, Context API

Backend: Node.js, Express.js

Database: MongoDB (Mongoose ODM)

Image Hosting: Cloudinary

Authentication: JWT (JSON Web Token)

Styling: CSS / Tailwind (if used)

# ⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/Shivi-5871/Chat-Application.git
cd chat-application

2️⃣ Install dependencies
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

3️⃣ Setup environment variables

Create a .env file inside the backend folder and add:

MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret_key
PORT=your_local_port_address

4️⃣ Run the application
# Run backend
cd backend
npm start

# Run frontend
cd ../frontend
npm start

5️⃣ Open in browser
http://localhost:port_address
